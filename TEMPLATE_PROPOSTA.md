## 🚀 Como Usar o Template

### Passo 1: Criar Nova Proposta
Clique em **"Nova Proposta"** no dashboard

### Passo 2: Dados do Cliente (OBRIGATÓRIO)
Preencha:
- Nome do cliente
- Nome da empresa
- Upload da logo

### Passo 3: Revisar e Editar Template
Todos os outros campos já estão preenchidos! Apenas:

**✏️ Campos que você deve sempre editar:**
- [ ] Nome do cliente
- [ ] Nome da empresa  
- [ ] Logo do cliente

**💰 Campos que você pode editar conforme o cliente:**
- [ ] **Preços dos planos** (destacados em verde - R$ 2.500 e R$ 3.000)
- [ ] Funcionalidades específicas de cada plano
- [ ] Título da proposta (se for serviço diferente)

**📝 Campos opcionais:**
- [ ] Estratégia (já tem texto profissional)
- [ ] Descrição (já tem texto padrão)
- [ ] Adicionar ou remover planos

### Passo 4: Adicionar Criativos
- Upload do criativo principal (1080x1350)
- Upload de exemplos para carousel

### Passo 5: IA para Textos Extras
- Use "IA Ajudar" para Resultados Esperados
- Use "IA Ajudar" para Notas Personalizadas

### Passo 6: Salvar!
✅ Proposta profissional pronta em minutos!

## 💡 Dicas de Personalização

### Para Clientes Pequenos:
- Mantenha o **Plano Essencial** (R$ 2.500)
- Remova ou ajuste o Plano Profissional se necessário
- Foque nas funcionalidades básicas

### Para Clientes Médios:
- Destaque o **Plano Profissional** (R$ 3.000)
- Mantenha ambos os planos para dar opções
- Adicione mais funcionalidades personalizadas

### Para Clientes Grandes:
- Ajuste os preços para cima (ex: R$ 4.000, R$ 5.500)
- Adicione um terceiro plano "Premium"
- Inclua serviços extras como:
  - Gestão de múltiplas redes
  - Produção de conteúdo exclusivo
  - Relatórios avançados

## 🔄 Editando Propostas Existentes

Quando você **edita uma proposta existente**, os dados já salvos aparecem preenchidos (não o template padrão).

Isso permite que você:
- Faça ajustes em propostas antigas
- Duplique conceitos que funcionaram
- Mantenha histórico de mudanças
