## 3️⃣ Galeria de Criativos com Carousel

### O que foi feito:
Criei uma seção de criativos linda e profissional com:

#### **A) Criativo Principal do Cliente (1080x1350)**
- Um criativo grande e fixo
- Formato vertical ideal para Instagram
- Destaque com borda verde limão
- Badge "Criativo do Cliente"
- Fica fixo enquanto você rola a página

#### **B) Carousel de Exemplos**
- Múltiplas imagens de trabalhos anteriores
- Mostra **3 criativos por vez**
- **Scroll automático** a cada 3 segundos
- Botões de navegação (setas)
- Indicadores de posição (bolinhas)
- Efeito hover nas imagens

### Como adicionar criativos:

1. **Criativo Principal:**
   - Arraste e solte UMA imagem 1080x1350
   - Este será o criativo em destaque do cliente

2. **Carousel de Exemplos:**
   - Arraste e solte MÚLTIPLAS imagens
   - Podem ser trabalhos de outros clientes
   - Exemplos do mesmo nicho
   - Remove imagens clicando no X

### Layout Visual:

## 📱 Como Usar - Passo a Passo

### Criar Nova Proposta:

1. **Clique em "Nova Proposta"**

2. **Preencha os dados do cliente:**
   - Nome e empresa do cliente
   - Logo do cliente

3. **Os dados da Zeri já estão preenchidos!**
   - Só ajuste se necessário

4. **Adicione criativos:**
   - Upload do criativo principal (1080x1350)
   - Upload de múltiplas imagens para carousel

5. **Use a IA:**
   - Clique em "IA Ajudar" nos Resultados Esperados
   - Clique em "IA Ajudar" nas Notas Personalizadas
   - Edite os textos gerados se quiser

6. **Salve e compartilhe!**

### Na Página da Proposta:

- O criativo principal fica fixo e destacado
- O carousel mostra 3 exemplos por vez
- Auto-scroll mostra todos os trabalhos
- Cliente navega com setas se quiser

## 💡 Dicas de Uso

1. **Para Criativos:**
   - Use imagens de alta qualidade
   - Criativo principal: 1080x1350 (formato Instagram)
   - Carousel: pode ser qualquer tamanho, mas idealmente 4:5

2. **Para Textos IA:**
   - Gere primeiro com IA
   - Depois personalize para o cliente
   - Adicione números e dados específicos

3. **Para Carousel:**
   - Adicione 4-6 imagens para melhor efeito
   - Selecione trabalhos do mesmo nicho do cliente
   - Mostre diversidade de estilos
